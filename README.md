# asteroid-geospatial-microservices-bachelor-thesis
Academic project implementing a scalable microservices platform for geospatial asteroid data analysis and visualization

## Overview

This project was developed as part of my Bachelor's thesis and focuses on building a microservices architecture for processing, integrating, and visualizing geospatial asteroid data.

Technologies used include Node.js, MongoDB, SQL databases, Cesium for 3D visualization, and Pentaho Kettle for ETL processes.
